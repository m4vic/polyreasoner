# PolyReasoner Experiments Index

This directory stores reproducibility assets and benchmark suites that support PolyReasoner claims.

## Active Validation Packs

- `paper4_polyreasoner_validation/`
  - Main migrated validation bundle (MoE puzzle benchmark, frontier benchmark, orchestration scripts, security-gatekeeper pipeline code, and paper figures).
- `DIVERSITY_VS_DEPTH.md`
  - Core research framing note for Polyreasoner claims (what is proven, what is pending, and how to report diversity vs depth results).

## Notes

- Large training artifacts are intentionally excluded from this repo area:
  - model checkpoints
  - generated output blobs
  - raw dataset dumps

- The goal here is portable reproducibility scripts and result snapshots, not heavyweight model storage.
